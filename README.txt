В данной задаче предполагается, что у вас уже есть класс реализующий некоторую логику работы игры в пятнашки. Теперь вам предстоит его расширить.

Сделайте класс Game2 со следующей функциональностью:
1.Game2 унаследован от оригинального Game;
2.Добавьте в новый класс функционал рандомизации стартовой последовательности;
3.Добавьте в новый класс функционал определения успешного завершения игры;

Сделайте класс Game3 со следующей функциональностью:
1.Game3 унаследован от Game2;
2.Добавьте в новый класс функционал отслеживания и сохранения истории движений;

Требования:
1.Грамотная объектная декомпозиция и абстрагирование - разделение на классы, выделение полей и методов классов;
2.Обработка некорректных данных при создании объекта.
3.Использование конструкторов для создания объектов;
4.Корректное разбиение проекта на файлы (модули);
5.Соблюдение соглашений по именованию и форматированию кода: корректное именование модулей с кодом и заголовочных файлов, классов, методов и полей классов, переменных.
6.Корректная обработка исключений.
